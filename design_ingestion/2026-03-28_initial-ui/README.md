# Design Ingestion Snapshot

Date: 2026-03-28

Purpose: Initial Dravix UI design ingestion system

Source: Bolt / external UI projects to be integrated later

Notes:

- This folder stores UI snapshots before they are integrated into the production frontend.
- `design_source` holds raw UI exports.
- `stripped_components` holds reusable React components.
- `design_tokens.json` stores design variables such as colors, fonts, and spacing.
