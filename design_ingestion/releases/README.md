# Dated Design Intake Releases

Use this directory to archive each incoming design package by date.

## Naming

- Create folders using `YYYY-MM-DD`.
- If more than one package arrives on the same date, suffix with a short label:
  - `2026-03-28-bolt-refresh`
  - `2026-03-28-brand-pass`

## Required Structure

Each dated folder should contain:

- `manifest.md`
- `bolt_ui/`
- `branding_guide/`
- `assets/logos/`
- `assets/icons/`
- `assets/images/`

## Workflow

1. Create a new dated folder.
2. Drop the new source files into that folder without modifying them.
3. Record provenance, author, and notes in `manifest.md`.
4. Promote selected assets into the top-level working staging folders only after review.
