# Dravix v0.3.0
Phase-2 Engineering Prototype

Dravix v0.3.0 represents the first stable engineering release of the Dravix fire-risk screening system.

The system provides an interpretable machine-learning-driven screening signal designed to support early-stage material evaluation before physical fire testing.

## Key Features

• deterministic machine-learning inference pipeline  
• tree-based ensemble model (RandomForest-style)  
• 1,771-material training dataset  
• 15-feature model representation  
• interpretability via treeinterpreter feature contributions  
• variance-based confidence indicators  
• stateless FastAPI inference service  
• live API documentation and interactive demo interface  

## System Scope

Dravix is a decision-support tool designed for early-stage material prioritization.

The system does NOT:

• replace standardized fire testing  
• certify fire resistance  
• perform regulatory decision-making  

Instead, it enables engineers to compare candidate materials and focus laboratory testing on the most promising options.
